# GearGuard: The Ultimate Maintenance Tracker

GearGuard is a web-based maintenance management system to track assets and manage maintenance requests efficiently. The system connects Equipment, Teams, and Requests seamlessly.

---

## Dashboard Overview

- **Request Type:** Select either Equipment or Work Center.
- **Assign Technician:** Choose a technician from the dropdown list.
- **Create Request:** Click to add a new task to the Kanban board.
- **Kanban Board Columns:** New | In Progress | Repaired | Scrap
- **Drag & Drop:** Move tasks between columns; badges update automatically.
- **Filters:** Filter tasks by Team/Technician or Request Type.
